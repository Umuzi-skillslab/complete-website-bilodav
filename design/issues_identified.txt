------------------------------------

2. Missig <meta charset="UTF-8> - Character coding is undefined if not applied.
3. Missing <meta name="viewport"> - The site will not be responsive on mobile.
4. Missing <meta name="description> - No SEO page description.
5. No CSS files exists in the repository leaving the pages completely unstyled.
6. All the structural elements of the HTML use <div> instead of semantic HTML elements.
7. There is no navigation bar.
8. There are no aria-label, aria-current, or role attributes anywhere.
9. Footer is unstyled.
10. There are no Open Graph tags for social sharing.
