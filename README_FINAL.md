# Portfolio Website Overview

This website is a portfolio website template. The website showcases a home page where a short into is given. An about page where more in depth information is given about the person. A Projects Page that Showcases the different past projects of the person. Lastly a contact page that allows a user to contact the profile owner easily via a form or contact buttons. The website is optomized for all screen sizes and is fully responsive across multiple devices

## Final HTML Structure

The final HTML Structure includes meta tags for better SEO and accesibility as well as OG tags.

It includes all aria-labels and and semantic elements and only two styling divs - one for grid layout with the class .container-grid and one for flex layout with the class .container-flex

## CSS Styling Approach

I used a root styling to keep my code concise, and just reusing the variables where needed. I used media queries to make the website responsive across all screen sizes. I used transitions and animations for interactivity

## Accessibilty improvements

I added aria labels for when inputs are focused and I added aria-labels for all non semantic elements
